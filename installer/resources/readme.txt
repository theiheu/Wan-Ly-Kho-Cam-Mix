TỔNG QUAN
=========
Phần mềm Quản lý Cám - Trại Gà là giải pháp quản lý toàn diện cho các trại chăn nuôi gà, 
giúp theo dõi lượng cám tiêu thụ hàng ngày, quản lý tồn kho và tạo báo cáo chi tiết.

TÍNH NĂNG CHÍNH
===============
✓ Quản lý cám hàng ngày theo khu vực và ca làm việc
✓ Theo dõi tồn kho nguyên liệu với cảnh báo thông minh
✓ Quản lý công thức cám và mix chuyên nghiệp
✓ Báo cáo và phân tích chi tiết
✓ Giao diện hiện đại, dễ sử dụng
✓ Xuất dữ liệu Excel
✓ Lịch sử và theo dõi xu hướng

YÊU CẦU HỆ THỐNG
================
- Hệ điều hành: Windows 10 (64-bit) hoặc mới hơn
- RAM: Tối thiểu 4GB, khuyến nghị 8GB
- Dung lượng ổ cứng: 200MB trống
- Quyền quản trị viên để cài đặt

HƯỚNG DẪN CÀI ĐẶT
=================
1. Chạy file cài đặt với quyền quản trị viên
2. Làm theo hướng dẫn trên màn hình
3. Chọn thư mục cài đặt (mặc định: Program Files)
4. Chọn các tùy chọn bổ sung (Desktop shortcut, Start Menu)
5. Hoàn tất cài đặt và khởi chạy ứng dụng

HƯỚNG DẪN SỬ DỤNG NHANH
=======================
1. Khởi động ứng dụng từ Desktop hoặc Start Menu
2. Thiết lập công thức cám mặc định trong tab "Công Thức"
3. Cập nhật tồn kho trong tab "Tồn Kho"
4. Nhập dữ liệu cám hàng ngày trong tab "Tổng quan"
5. Xem báo cáo và lịch sử trong các tab tương ứng

GỠ CÀI ĐẶT
===========
Để gỡ cài đặt phần mềm:
1. Vào Control Panel > Programs and Features
2. Tìm "Phần mềm Quản lý Cám - Trại Gà"
3. Nhấp "Uninstall" và làm theo hướng dẫn
4. Chọn có/không xóa dữ liệu người dùng

LƯU Ý QUAN TRỌNG
================
- Luôn sao lưu dữ liệu quan trọng trước khi cập nhật
- Đảm bảo có quyền ghi vào thư mục cài đặt
- Tắt phần mềm antivirus tạm thời nếu gặp vấn đề cài đặt
- Liên hệ hỗ trợ nếu gặp bất kỳ vấn đề nào
